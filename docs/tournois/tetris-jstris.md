# Tournament: Tetris - jstris Maps Challenge

**Date:** Mardi 21  
**Horaire:** 18h00 - 21h30  
**Mode:** Maps

---

## 🎮 À propos du tournoi

Bienvenue dans le tournoi **Tetris jstris**! Ce tournoi vous propose de relever des défis de maps Tetris sélectionnées par les organisateurs. L'objectif est simple : **compléter chaque map dans le minimum de temps possible**.

---

## 📋 Règles

### Format

- Mode **Maps uniquement** sur jstris
- Les maps sont sélectionnées et fournies par les organisateurs
- Chaque map doit être complétée

### Scoring

- **Plus vite = Plus de points** : Votre temps de réalisation est converti en score
- **Punitif si non-terminé** : Vous devez terminer chaque map pour scorer des points
- Le système de scoring détaillé sera révélé avant le début

### Déroulement

- Les manches durent **15 minutes**
- Vous pouvez recommencer une map autant de fois que vous le souhaitez pendant ce créneau
- Plusieurs maps seront proposées pendant la soirée
- Votre meilleur temps pour chaque map compte

---

## ⚙️ Préparation Requise

### 1. Créer votre compte jstris

- Rendez-vous sur [jstris.jezevec10.com](https://jstris.jezevec10.com)
- Créez un compte gratuit avec votre pseudo
- **Important:** Configurez vos touches de contrôle dans les paramètres

### 2. Configurer vos touches

- Accédez aux paramètres de contrôle (Settings > Controls)
- Configurez les touches selon votre préférence (QWERTY, AZERTY, manette, etc.)
- **Conseil:** Testez vos touches avant le tournoi pour être à l'aise

### 3. Comprendre le mode Maps

- Le mode Maps vous permet de jouer des défis pré-configurés
- Consultez la [bibliothèque de maps](https://jstris.jezevec10.com/maps) pour des exemples
- Vous pouvez vous entraîner sur d'autres maps en attendant le tournoi

---

## 🎯 Conseils pour Réussir

- **Pratiquez avant:** Familiarisez-vous avec jstris et le mode Maps
- **Optimisez vos touches:** Plus elles sont naturelles, plus vous serez rapide
- **Concentrez-vous:** Chaque seconde compte!
- **Restez patient:** Si une map est difficile, prenez un moment et recommencez

1. **18h00 - 18h15:** Début du tournoi - Première manche
